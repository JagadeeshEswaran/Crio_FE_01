Positioning in CSS

What is the use case ?
  • When we want to "control the layout and arrangement of elements" on a web page.
Let's understand CSS Positioning
  • Position property in CSS helps us to manipulate the location of an Element.
  • There are 5 different strategies we can follow
    1. Static
    2. Absolute
    3. Relative
    4. Fixed
    5. Sticky
  • Position is a non-cascading (non0-inherited) property.
  • Position property goes hand in hand with Positioning attributes like left, right, top, bottom and Z-index properties.
Display Properties
  • Since every element on the page is a rectangular box, the display property determines how that box behaves.
  • Some common values,
    • inline -> Make the box (or Block element) an in-line element
    • block -> Default for all block elements & can Make the box (or In-line) element and Block Element.
    • inline-block -> Make the box a block element which would behave like in-line element.
    • flex -> Makes the element behave flex box container.
    • grid -> Makes the element behave flex grid container.
