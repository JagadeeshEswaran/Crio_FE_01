CSS Selectors

  • Selectors in CSS are used to find elements we want to Style
  • These are multiple selectors which can be used to pick Specific or Multiple elements
  • Common Selectors,
    1. Universal Selector (*) - Selects Everything in the HTML Document.
    2. Element Selector - Selects elements with same name
    3. Class Selectors (.) - Selects elements with Same Class attribute value.
    4. ID Selectors (#) - Selects elements with particular ID
  • CSS is written inside curly braces "{}" and in 'key: value' pairs & Ends with semi-colon
