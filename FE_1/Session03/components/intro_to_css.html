Introduction to CSS

  • Stands for Cascading Style Sheets
  • It is a Language to define styles like font, size, spacing, color, etc to HTML elements
  • There are ways to specify styles,
    1. In-line - Using "Style" attribute
    2. Internal - Unsing "Style" tag.
    3. External - Specify styles in a separate ".css"
  • CSS is written inside curly braces "{}" and in 'key: value' pairs & Ends with semi-colon
