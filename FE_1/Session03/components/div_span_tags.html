div & span tags

<div> Tag

  • The <div> elements is often used as a container for other HTML elements.
  • We can say for "Grouping"
  • <div> have no require attributes
  • <div> tags are "Block Elements"

<span> Tag

  • The <span>elemts is used to mark-up a part of Text or a part of a document.
  • We can say for "Highlighting"
  • <span> have no require attributes
  • <div> tags are "In-line Elements"
