Psuedoe-Class Selectors

  • A psuedo-class is used to define a special state of an element.
  • These selectors can be used based on Specific Conditions like,
    1. Bringing cursor over (hover) an element.
    2. Clicking on an Element (Active)
    3. Visited Links.
  • We can use (:) for access psuedo-selectors.
