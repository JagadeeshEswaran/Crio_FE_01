Activity - Creating a Navbar with Bootstrap

Step 1 - Setup our Components

  • Create simple HTML document with <nav> tag, create a container for Navbar.
  • Set H2 - Posts, ul, li - Products, Services, Contact us
  • After Installing Bootstrap, the navbar would look like this.
  • Add the class "navbar" to your component to make it look like navbar.
  • By adding container-fluid class, we could specify "width: 100%"
  • For all the colors to work, we need to specify the color scheme. Use class navbar-light to the nav component.

Step 2 - Styling

  • Now we'll style our ul - components.
  • For this Bootstrap provides us,
    • navbar-nav (for wrapping the "nav" item)
    • nav-item (for the "nav li")
    • nav-link (to wrap the links inside "li")
  • Apply the navbar-nav class to "ul" and nav-item class to list items under it.
  • Wrap the content inside the "li" into its own anchor tag with the class nav-link
  • We can use href="#" to create empty link.
