Introduction to Bootstrap

  • Bootstrap is a CSS Framework by Twitter.
  • It allows us to leverage pre-written CSS classes, which we can directly use out-of-the-box.
  • It also gives us short-hand notations for common CSS like,
    • flex, grid, margin, padding, etc.
  • Bootstrap has lot of built-in components with their own CSS
    • (and functionality like JS)
  • Common components : Navbar, Carousel, Accordion, Spinner etc
  • Refer @ Bootstrap Home page

Bootstrap Documentation

Structure of Bootstrap

  • Bootstrap primarily works on the concepts of CSS Class selectors.
  • We can attach multiple classes to any HTML elements, seperated by spaces.
  • The structur can be mapped down to 4 main parts.
    1. Layout - Containers, Grid, Columns, Breakpoints, Z-Index, etc.
    2. Content - Typography, images, etc.
    3. Components - Pre built Components like Cards, navbar, carousel, button, etc.
    4. Utilities - Your plain CSS which had been writing up till now.
