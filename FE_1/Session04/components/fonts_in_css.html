Using Fonts with CSS

  • Introducing Typography
  • A typeface or font-families is a collection of letters, numbers, symbols (also called as glyphs)
  • Ex.: Serif, Sans-Serif, Times New Roman, Comic Sans, Arial, etc.
  • Mostly used font-families are, serif & sans-serif
  • Serif Fonts
    • Small stroke attached to the end of letters
    • Mostly used in books and text
  • Sans-Serif are more smooth and look modern
    • Used in headings and app UIs
  • We can specify the font's Size, Weight and Style using "font-*" properties.
