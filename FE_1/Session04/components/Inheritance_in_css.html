Inheritance in CSS

  • Some properties are inherited down the HTML while others are not.
  • Styles that are inherited are generally related to the styling of the document text.
    • Ex.: Fonts & Colors
  • Styles that are not inherited are generally related to the styling of the document text.
    1. Paramets for Positions : Sizes, Margins, Background & frames.
    2. Ex.: width, height, border, padding, margin, position, etc.
  • Don't confuse this with the fact that block elements take all the available width.
