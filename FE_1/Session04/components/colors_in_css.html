Using Colors in CSS

  • Colors in HTML & CSS are specified in a few different ways.
    • Prefefinded Color Name => ~143 colors (Ex.: background-color: "red";)
    • RGB - Red Greed Blue. (ex.: rgb(0,0,255))
    • RGBA - Red Greed Blue Alpha Channel. (ex.: rgba(0,0,255, 0.5))
