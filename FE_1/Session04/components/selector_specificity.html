Selector Specificity

  • When two CSS rules are in Conflict, it is resolved by using CSS Specificity
  • More specific the selector, higher the preference.
  • Hierarchy,
    1. In-line Styling
    2. ID Selectors
    3. Class Selectors
    4. Element Selectors
  • When selectors of equal Specificity are in conflict - Lower (or Latest) rule in the Stylesheet wins.
