Creating Navbar with Bootstrap - Part 2

Step 1 -> Creating a target
  • Wrap the ul inside a div -> which will become our target to toggle on button click.
  • Give it a class "navbar-collapse" to give it the positioning it needs.(i.e) Under the Navbar logo
  • Add an "id = navBarTarget", which will use later to connect the button.
Step 2 -> Adding a Button & Functionality
  • We can use "data-*" attribute, which are custom attributes.
  • We can point it to any element we want. We're going to target "id = navBarTarget".
  • Add a class "collapse" to the "navBarTarget" div
  • Add attributes -> data-bs-toggle="collapse" and "data-bs-target="#navBarTarget" to the button.
  • This will add a collapse Functionality to the menu targeting "navBarTarget" div.
Step 3 -> Adding Hamburger Menu
  • Final step is to add a Hamburger icon using Bootstrap class.
  • Add a span tag within the button element, and add "navbar-toggler-icon" class to it.
  • Our Navbar is ready for mobile !
Step 4 -> Making Navbar responsive
  • Our navbar looks great for mobile screen, what about larger screens.
  • To make it responsive, first we need to specify break point & behaviour
  • Breakpoints in Bootstrap are defined as ".class-{breakpoint}-{value}"
  • On large screens, our navbar should be spread out its items show it.
  • To achieve that, we can add "navbar-expand-*"
  • We can spcify a class "navbar-expand-lg" at outer div with "navbar" class.
