Fitting Images inside div

  • Often times, Cards have images in on part & the content in other Part of it.
  • We can achieve this effect by "nesting" an "img" tag inside "div" tag
  • Then set "width and height" of the image to "100%"
  • Since % are relative units to parent, the image will take up the size of the parent div

Adding a Hero Image

Step 1 -> Creating a target
  • Create a header element and nest the existing nav inside it.
  • Add an "img" nested inside a new "div" above the "nav" element.
  • Define a height of "16rem" for the "header" and make the image fir 100% of the height and width of the parent.
Note -> How to prevent Image Squishing ?
  • "Object-fit" -> prevents squishing. i.e Maintains aspect ratio.
  • "object-fit: contain" -> Resize image to fit the given parent dimensions "without clipping"
  • "object-fit: cover" -> Does the same, but image is "clipped"
