CSS Grids

  • CSS Grid is a Two dimensional grid-based layout system.
  • It helps place the rectangular elements on a web page in complex manners.
    • Elements of Grid System
    1. Grid Container (Parent)
    2. Grid Item (Children)
    3. Grid Rows (Horizontal parts)
    4. Grid Columns (Vertical parts)
    5. Grid Lines - Which divide the rows and columns
    6. Grid Cells - Spece between two adjacent rows & grid lines
    How Grid System Works ?
    • The Grid Container (parent) forms a wrapper element for all the Grid Items (Children) under it.
    • All direct descendant elements of the Grid container are known as Grid Items.
    • We can define it with "display: grid" CSS Property
    Rows and Columns in Grid System
    • To make our linear grid two-dimensional, we need to define the "number of rows & columns"
    • We can make use of, "grid-template-*" properties of the grid container.
