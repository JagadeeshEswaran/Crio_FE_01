What is Anchor Tags ?

  • HTML < a > defines a Hyperlink.
  • You can Click on it to jump to a new page or a Section
  • < a > tag has on important attribute -> 'href'
  • 'href' -> indicates the link's destination.
  • The content for 'a' tag can be taxt ot even Image
