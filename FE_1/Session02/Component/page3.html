Images in HTML

  • Images can be displayed on a Webpage using the <img> tag

  • The image tags need one important attribute to display the image 'src'

How to display images in our Webpages ?

height & width attributes

Simply Using img tag with src attribute

Using 'height' & 'width' attributes => height="200" width="200"

Using 'height' & 'width' attributes => height="100" width="100"

What of the used Image is not available to display ?

We can make use of 'alt' attribute in the <img> tag

Using 'height' & 'width' attributes => height="200" width="200"

Using 'height' & 'width' attributes => height="100" width="100"

What are the Image Formats available ?

  • While working with images, it's important to understand the different file types.
  • Knowing when to use what format will lead to better image quality and faster site loading
  • it's a good idea to learn about JPG, GIF, PNG, SVG and Vetor image formats.
