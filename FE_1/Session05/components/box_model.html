Box Model with HTML & CSS

  • Every element in web designing is a rectangular box
  • CSS Box model contains 4 parts which are (from out to in)
    1. Padding
    2. Margin
    3. Border
    4. Content
  • The width & height properties of CSS controls the content area of the box.

Padding & Margin => Activity

  • Padding
    • Inner-Spacing of an Element wrapping only thecontent.
    • Crontrolled using padding-* properties
  • Margin
    • Outer-Spacing of an element wrapping content + padding + border
    • Decides distance between the current box & other box.
    • Controlled using margin-* properties.
