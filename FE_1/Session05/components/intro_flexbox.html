Introduction to FlexBox

  • Flexbox is used to layout, align and distribute space among items in a container.
  • It is useful when the size of the container/item is unknown & dynamic.
  • It is well suited for application components like inside the navbar - forms - cards, etc.
  • Flexbox is made up of two main conponents.
    • Flexbox Parent (Container)
    • Flexbox Children (Item)
  • Important terms to remember :
    • Main Axis => Default is Horizontal (We could change it)
    • Cross Axis => Default is Vertical (We could change it)
