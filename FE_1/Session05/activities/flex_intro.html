Activity - Flexbox Intro

  • Create a section with class name of "navbar" (container)
  • Create a 3 child items (Children)
  • Group "navbar" & child items with "logo" & "links" classname containers.
  • Set display property as "flex" to both "navbar" & "links" containers
  • Give some left & right padding of 25px to "navbar" container.
  • Set "justify-content" property as "space-between"
  • Set "width" property to "links" container as 50%. This will give 50& width of parent container.
  • Then spread items in "links" container by setting justify-content propertu to "space-around"
